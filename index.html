Introduction

Velonic is a fully featured premium admin template built on top of awesome Bootstrap v5.3.0. It's crafted using modern web technologies HTML5, SCSS and React (Typescript). The theme includes beautiful dashboard, many ready to use hand crafted components, widgets, pages, etc. The components can be used very easily on any page. The theme is fully responsive and customizable.

It is extremely powerful and adaptable, making it ideal for web development. applications including CRM, CMS, admin panels, dashboards, etc. It is fully responsive and has a perfect look and feel on mobiles and tablets. Also, we tried to implement this by Keeping the concept of modularity in mind, this enables you to reduce the development time and cost.

This documentation is guidelines for all users who can even be a beginner or experienced web developer.

Additional features :

  • Lazy Loading(React.lazy)
  • Custom Hooks

Structure

When you extract the zip file you received after purchasing, you will find the following files and folders: 

                                    
                                      ├── Documentation
                                      ├── Velonic
                                      │   ├── package.json                => Package json file
                                      │   ├── README.md                   => Create react app read me file
                                      │   ├── public
                                      │   │   └── favicon.ico
                                      │   │   └── images/logo
                                      │   ├── src/
                                      │   │   └── assets                    
                                      │   │       └── fonts               => fonts files
                                      │   │       └── images              => images
                                      │   │       └── scss                => scss style files
                                      │   │   └── common                  => api,context,helpers
                                      │   │   └── components              => Reusable components
                                      │   │   └── constants               => App level constants
                                      │   │   └── hooks                   => Hooks - Custom hooks
                                      │   │   └── layouts                    
                                      │   │       └── HorizontalLayout    => Horizontal Topbar and top navbar, footer and right side bar
                                      │   │       └── VerticalLayout      => Left side navbar with topbar, footer and right side bar
                                      │   │   └── pages                   => All the pages available in the theme
                                      │   │   └── routes                  => Application router implementation
                                      │   │   └── utils                   => Common utilities
                                      │   ├── App.tsx                      => Main application component
